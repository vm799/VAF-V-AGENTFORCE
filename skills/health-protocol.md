# Health Protocol

Use this skill for daily health logging, the 5x5 Goggins physical protocol, and weekly trend reviews. The body is the foundation — everything else degrades without it.

## The 5x5 Goggins Physical Protocol

Five exercises, five days per week. No negotiation. No modification downward. Scale up only.

### The 5x5 Weekly Structure

| Day       | Focus         | Primary Work                          | Duration |
|-----------|---------------|---------------------------------------|----------|
| Monday    | Push + Run    | Push-ups, Dips, Bench, OHP + 5K run  | 60-75 min |
| Tuesday   | Pull + Core   | Pull-ups, Rows, Deadlift + Core circuit | 60 min |
| Wednesday | Run + Mobility | 5-10K run + 30 min stretching/yoga   | 60-75 min |
| Thursday  | Legs + Core   | Squats, Lunges, RDLs + Core circuit   | 60 min |
| Friday    | Full Body + Run | Compound lifts + 5K run             | 75 min |
| Saturday  | Active Recovery | Long walk (60 min+), stretching      | 60 min |
| Sunday    | REST          | Full rest. No guilt. Recovery is work. | — |

### The 5 Exercises (Non-Negotiable Minimums)

These are the floor, not the ceiling:

1. **Push-ups**: 100/day (broken into sets throughout the day if needed)
2. **Pull-ups**: 50/day (use bands if needed, no ego)
3. **Squats**: 100/day (bodyweight counts)
4. **Core**: 200 reps/day (mix of crunches, leg raises, planks in seconds)
5. **Run**: 3x per week minimum, 5K minimum distance

### Progressive Overload Tracking

```
| Week | Push-ups | Pull-ups | Squats | Core Reps | Run Distance | Run Pace |
|------|----------|----------|--------|-----------|--------------|----------|
| W1   |          |          |        |           |              |          |
| W2   |          |          |        |           |              |          |

```

Rule: Every week, increase at least ONE number. Never decrease.

## Nutrition Protocol

### Daily Targets

| Metric | Target | Tracking |
|--------|--------|----------|
| Protein | 2g per kg bodyweight (minimum 150g) | Per meal |
| Water | 3+ litres | Track by bottle (750ml x 4) |
| Meals | 3 main + 1 snack | Timed |
| Meal Window | 08:00 - 20:00 (12hr window) | Clock |
| Vegetables | Every meal must include veg | Checkbox |

### Foods to Prioritize
- Protein: Chicken, fish, eggs, Greek yogurt, whey, lentils
- Carbs: Rice, oats, sweet potato, fruit
- Fats: Avocado, nuts, olive oil
- Avoid: Processed food, excessive sugar, alcohol (limit to 1x/week max)

## Sleep Protocol

### Targets
- **Duration**: 7-8 hours minimum
- **Consistency**: Same bed time +/- 30 minutes every night
- **Quality**: Aim for <15 min to fall asleep, <2 wake-ups

### Sleep Hygiene Rules
1. No screens 30 minutes before bed
2. Room temperature: 18-19°C
3. No caffeine after 14:00
4. Last meal 2+ hours before bed
5. Bedroom is for sleep only — no working in bed
