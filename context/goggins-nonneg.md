# Goggins Non-Negotiables — Daily Scoring System

Last updated: 2026-03-27

---

## The 5 Non-Negotiables

Every single day. No days off from scoring. Rest days still score (recovery counts for BODY, reading counts for LEARN).

| # | Non-Negotiable | What It Means | Max Score |
|---|---------------|---------------|-----------|
| 1 | **BODY** | Complete the 5x5 physical protocol. Workout, movement, physical effort. | /10 |
| 2 | **BUILD** | Ship one thing. Code, config, system, feature. Something tangible exists that did not before. | /10 |
| 3 | **LEARN** | One lesson consumed + one insight extracted. Course module, chapter, article with a note. | /10 |
| 4 | **AMPLIFY** | One content piece created or published. LinkedIn post, Substack draft, thread. | /10 |
| 5 | **BRIEF** | Morning briefing + evening review completed. Bookend the day with intention and reflection. | /10 |

**DAILY MAXIMUM: 50 points**

---

## Scoring Rubric

### BODY (/10)
| Score | Criteria |
|-------|----------|
| 0 | No physical activity |
| 2 | Light movement only (walk <30 min) |
| 4 | Moderate activity (walk 30+ min or light workout) |
| 6 | Solid workout (45+ min, broke a sweat) |
| 8 | Full 5x5 protocol completed (all minimums hit) |
| 10 | 5x5 protocol exceeded + ran 5K+ or hit a new PR |

### BUILD (/10)
| Score | Criteria |
|-------|----------|
| 0 | Nothing shipped or created |
| 2 | Planning or research only (no tangible output) |
| 4 | Started building but did not finish/ship |
| 6 | Shipped one small thing (config, fix, minor feature) |
| 8 | Shipped something meaningful (feature, system, integration) |
| 10 | Shipped something significant AND it is live/deployed/usable |

### LEARN (/10)
| Score | Criteria |
|-------|----------|
| 0 | No learning |
| 2 | Passive consumption (scrolling, skimming) |
| 4 | Active consumption (read article, watched video with focus) |
| 6 | Consumed + extracted insight (wrote down what you learned) |
| 8 | Created permanent note in Obsidian with connections |
| 10 | Full synthesis: note created + content opportunity identified + applied to work |

### AMPLIFY (/10)
| Score | Criteria |
|-------|----------|
| 0 | No content created |
| 2 | Idea captured / outline started |
| 4 | Draft written but not published |
| 6 | One piece published (LinkedIn post or Substack section) |
| 8 | One piece published with strong engagement (10+ comments or high open rate) |
| 10 | Multiple pieces published OR one piece that drove inbound leads/subscribers |

### BRIEF (/10)
| Score | Criteria |
|-------|----------|
| 0 | No briefing at all |
| 3 | Morning briefing only |
| 5 | Morning briefing + set 3 priorities |
| 7 | Morning + evening review |
| 10 | Morning briefing + 3 priorities set + evening review with scores logged |

---

## Daily Thresholds

| Total Score | Rating | Meaning |
|-------------|--------|---------|
| 0-15 | **UNACCEPTABLE** | Wasted day. What happened? Fix it tomorrow. |
| 16-25 | **Below Standard** | Showed up but did not push. |
| 26-35 | **Standard** | Solid day. Met expectations. |
| 36-45 | **Strong** | Exceeded in most areas. Momentum building. |
| 46-50 | **GOGGINS** | Elite day. Crushed every dimension. Rare and earned. |

| Days at GOGGINS level (46+) | 0 |

### Streak Rules
- A day counts toward the streak if total score is **26+** (Standard or above)
- Below 26 breaks the streak
- Streak resets to 0 on a miss
- Track longest streak ever as motivation
