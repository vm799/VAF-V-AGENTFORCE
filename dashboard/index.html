One-Time Setup
Do Once
1
Open Terminal, go to your project

cd ~/Desktop/vaishali-agent-force

2
Install LaunchAgents (auto-start on login)

./vaf.sh start
Everything runs automatically every time you log in.

3
Set up phone access tunnel

./scripts/setup_tunnel.sh
Follow the prompts — installs Cloudflare tunnel so your dashboard works on your phone.

4
Check your .env file has these
# Required
VAF_TELEGRAM_TOKEN=your_bot_token
VAF_TELEGRAM_CHAT_ID=your_chat_id
VAF_OBSIDIAN_VAULT_DIR=/path/to/SecondBrain
# Optional (for AI summaries)
VAF_ANTHROPIC_KEY_1=sk-ant-...
After this — you never need to touch the terminal again. Everything starts at login, briefing lands in Telegram at 7am, evening update at 9pm.
